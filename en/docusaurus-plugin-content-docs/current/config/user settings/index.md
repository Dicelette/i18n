---
title: Per user 
---
The command ⁨`/user_setting`⁩ allow to configure parameters for each user.

It’s possible to configure :
- [Snippets](./snippets.md) : Reusable dices not tied to a character sheet or a model.
- [The link created by the contextual menu](../context_menu.md)
- [Attributes](./attributes.md) : A sub-system of statistics that can be used as shortcuts in roll formulas and that automatically resolve during the roll.
